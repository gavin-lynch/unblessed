# Example Ideas

Based on the current contrib examples (charts, gauges, maps, layouts), here are 10 new example ideas that would expand coverage and showcase features.

1. **System Overview Dashboard**
   - Grid layout with CPU/Memory gauges, disk usage bar, and log tail.

2. **Multi-Series Line + Sparkline Combo**
   - Shared data stream, line chart for long horizon, sparkline for recent window.

3. **Interactive Filters**
   - Key-controlled filter toggling series on/off in line/bar charts.

4. **Alarm/Threshold Visualization**
   - Gauge + line chart with threshold highlighting and blinking status box.

5. **Map + Status Pane**
   - World map markers with a table listing status/latency per region.

6. **Donut + Stacked Bar Comparison**
   - Donut shows percentages, stacked bar shows absolute values for same dataset.

7. **Live Log + Metrics**
   - Log widget streaming events alongside a gauge or line chart updated in sync.

8. **Carousel with Async Pages**
   - Demonstrate async page initialization and onAfterMove usage.

9. **Animated Theme Switcher**
   - Cycle through color themes and update widget styles live.

10. **Grid Stress Test**

- Dense grid of small widgets to validate layout, rendering, and performance.
